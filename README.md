# Hanix (HNX)

Premium landing page for the Hanix ERC-20 token on Base.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide Icons
- Wagmi + Viem
- Radix UI (Accordion)

## Getting started

\\\ash
npm install
npm run dev
\\\

Open [http://localhost:3000](http://localhost:3000).

## Configuration

Update placeholders in \src/lib/constants.ts\:

- Contract address
- BaseScan URL
- GitHub repository
- Documentation / social links
- Whitepaper PDF at \public/whitepaper.pdf\

## Scripts

- \
pm run dev\ — development server
- \
pm run build\ — production build
- \
pm run start\ — start production server
- \
pm run lint\ — ESLint
